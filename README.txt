=====================Assignment 1==============================
The assign was developed using Visual Studio 2015

The program loads Model from the 'form.txt' files located
in the COMP371_hw1 folder or in the Resources folder in the VS solution.

Functionalities:

Change model type:
Press:
P - for Points
W - for Lines
T - for triangles

Rotation:
Use directional to rotate object

Zooming In/Out:
Press and hold left mouse button and move the mouse foward and
backwards.
